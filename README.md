# Anton Verenitch Portfolio

A lightweight GitHub Pages portfolio inspired by the HUD-style reference at [jragni.github.io](https://jragni.github.io/), customized around Anton Verenitch's product leadership profile.

## Files

- `index.html` - the site content
- `styles.css` - visual design and responsive layout
- `script.js` - small header behavior
- `.nojekyll` - tells GitHub Pages to serve the static files as-is
- `.gitignore` - keeps local PDF exports out of the public Pages repo

## Publish On GitHub Pages

1. Create a public GitHub repository named `ton87.github.io`.
2. Add these files to the repository root.
3. In GitHub, open `Settings > Pages`.
4. Choose `Deploy from a branch`.
5. Select `main` and `/ (root)`, then save.
6. Your site will publish at `https://ton87.github.io/`.

For a project repository instead of a user site, publish the same way and GitHub will serve it at `https://<your-github-username>.github.io/<repo-name>/`.
